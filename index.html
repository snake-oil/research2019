github上に新規でリポジトリを作成する。 VScodeに先に作成したリポジトリを登録する。 gihub上で新規にリポジトリを作成する手順 githubにログインする。 上部のRepositoriesタブをクリックする。 移動したページで右上部のNewをクリックする。 移動したページで次の項目を入力する。 ・Repository name:resarch2019 ・Description:2019年 夏の自由研究 ・公開範囲:Public Create Repositoryをクリックしリポジトリを作成する。 表示されるhttpsとsshのコマンドラインを控えておく。 VScodeにリポジトリを登録する手順 ワークスペースにresarch2019というディクトりを作成し、そこへ移動する。 左帯にあるソース管理のアイコンをクリックする。 上部に出るアイコンによりディレクトリをリポジトリとして使えるように初期化する。 command + shift + p でコマンドラインを呼び出す。 コマンドラインにgit add Remoteを入力し、先に控えたhttpsとsshのコマンドラインを入力する。
