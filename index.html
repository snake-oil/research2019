github上に新規でリポジトリを作成する。 VScodeに先に作成したリポジトリを登録する。 リポジトリ上にファイルを作成しcommitする。 commitを繰り返し出来上がったファイルをGitHubへpushする。 gihub上で新規にリポジトリを作成する手順 githubにログインする。 上部のRepositoriesタブをクリックする。 移動したページで右上部のNewをクリックする。 移動したページで次の項目を入力する。 ・Repository name:resarch2019 ・Description:2019年 夏の自由研究 ・公開範囲:Public Create Repositoryをクリックしリポジトリを作成する。 表示されるhttpsとsshのコマンドラインを控えておく。 VScodeにリポジトリを登録する手順 ワークスペースにresarch2019というディクトりを作成し、そこへ移動する。 左帯にあるソース管理のアイコンをクリックする。 上部に出るアイコンによりディレクトリをリポジトリとして使えるように初期化する。 command + shift + p でコマンドラインを呼び出す。 コマンドラインにgit add Remoteを入力し、先に控えたhttpsとsshのコマンドラインを入力する。 commitする手順 リポジトリとして作成したディレクトリにいることを確認する。 ファイルを新規作成しindex.htmlという名前で保存する。 左帯のソール管理のアイコンをクリックする。 表示されるファイル名の横にあるプラスマークをクリックしリポジトリにファイルを追加する。 ファイルのstatusがUからAに変わる。 メッセージ欄に初めてのコミットと入力し、上部のチェックマークをクリックしcommitする。 pushする手順 ソース管理の選択コマンドの中からpsuhを選択する。 先に登録したGitHubの公開先を選択すればユーザとパスワードを聞かれので入力する。 GitHub上でリポジトリがアップされていることを確認する。 GitHubでリポジトリ->Settings->GitHub Pagesの項目へと進む。 Sourceの項目でmaster branchを選択する。 指定される https://snake-oil.gihub.io/resarch2019/ をブラウザで表示するとホームページが見られる。
